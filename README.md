# 🚀 8-Week SQL Challenge
**By Alex A.**

This repository contains my solutions for the **#8WeekSQLChallenge**, a series of 8 real-world SQL case studies designed by [Danny Ma](https://8weeksqlchallenge.com/). 

The goal of this challenge is to master complex SQL queries, data cleaning, and business analytical thinking across various industries including Food Tech, Finance, and Marketing.

---

## 🏗️ Repository Structure
Each folder is self-contained and includes organized scripts for both database schema setup (`schema.sql`) and final solutions (`solution.sql`).
